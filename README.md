
# Azure Terraform Examples

This repository contains useful examples of Terraform code for Azure

## Azure

| Name                | Description                                    |
|---------------------|------------------------------------------------|
| compute             | a compute instance                             |
| database            | a Postgres instance                            |
| network             | network and subnets                            |

### Usage

    az login
    cd environments/azure/dev
    terraform init
    terraform plan
    terraform apply
